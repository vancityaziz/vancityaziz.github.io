Mixture-of-Experts Ensemble for Skin Lesion Classification Aug 2025
APS360 Course Project | University of Toronto
ARCHITECTURE MoE Gating Network, EfficientNet (B0, B2, B3), MobileNet-V2
PERFORMANCE 82.5% Accuracy, 0.966 AUC, 74.8% Balanced Accuracy
  • Developed a Mixture-of-Experts (MoE) inspired ensemble using a top-k gating network to selectively weigh individual CNN experts.
  • Mitigated extreme class imbalance in the HAM10000 dataset using weighted cross-entropy loss and automated data augmentation (RandAugment, CutOut).
  • Implemented a two-stage training workflow: fine-tuning individual experts followed by frozen-backbone gating network optimization.
  • Achieved superior robustness over baseline SVMs, particularly in clinical recall for malignant classes.
